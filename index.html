Aprendendo HTML5

O HTML5 ele foi elaborado para ter semântica nas tags, ligando todas as programações já feitas, para assim aproveitar ferramentas de outros sistemas e programas já criados.

O HTML5 consiste em fazer a estrutura semântica do site, deixando qualquer tipo de forma e design para o CSS3 que é a linguagem de marcação usada para estilos no site.

O básico

O começo é usar a estrutura básica com o "!" no "vs code" e vai ser montada a estrutura do HTML inteira.

As tags básicas do HTML5 usadas são: html, head, body. Sempre lembre de fechar as tags.

Para fazer títulos basta usar <h1> e </h1> e escrever dentro da tag. Para sub-títulos deve mudar o número, quanto maior menor vai ser a fonte. A tag vai de <h1> até <h6>.
Também podemos definir dentro dos títulos os 'id', definindo um id você consegue criar uma âncora para sempre que quiser voltar até aquele tópico você ter atalhos para isso.
Utilizando em <a> na parte de href o '#' com o nome definido no 'id'.

Para fazer um paragráfo é so usar a tag <p> e </p> com a quebra de linha <br> para mudar a linha e <hr> para criar uma linha horizintal.

Para escrever um texto de forma que fique igual o texto que digitou no idle, use a tag PRE.

Também existe símbolos como: ©, 𝕔, ™, ℝ, entre outros.

Existe também emojis como: 🖖, 🤓, entre outros.

Hoje em dia também temos tags para dar ênfase e reforço como STRONG e EM.

Para usar imagem em HTML use a tag <img>, com ela na parte de src(source) entre as aspas você irá colocar o diretório da imagem que baixou e colocou dentro da pasta de onde está o arquivo html que criou.
Você também pode utilizar CTRL + ESPAÇO do seu teclado entre as aspas para já procurar as imagens que está dentro da sua pasta.

Para criar links na sua página use a tag <a> e dentro das aspas insira a url desejada para o cliente ser redirecionado.
Usando 'target' dentro da tag e '_blank' você consiguirá fazer com que o link abra uma nova aba sem que o seu site seja substituído na aba atual.

Cuidado para não usar tags obsoletas(não vai funcionar) por isso consulte aqui as tags que não estão sendo mais usadas!

Principais formatações

Negrito / Destaque

Nesta frase temos um termo em destaque usando a tag STRONG.

Itálico / Ênfase

Nesta frase temos um termo em ênfase usando a tag EM.

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto grande e pequeno

Podemos criar também textos grandes e textos pequenos com as tags BIG e SMALL.

Texto deletado

Podemos marcar um texto como excluído com a tag DEL.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois.

Texto sobrescrito e subscrito

Para escrever um texto sobrescrito utiliza-se a tag SUP.
Já para escrever a tag subscrita utiliza-se a tag SUB.

Código-fonte

O códifo fonte é feito com a tag CODE que é usado para todas as letras terem o mesmo espaçamento, ideal para códigos.

Citações simples

Para fazer uma citação usamos a tag <q> para citar alguma frase.
Como diria o pai de um amigo: o computador é um burro muito rápido

Citações completas

É usada a tag BLOCKQUOTE para fazer uma citação de algo retirado de algum outro lugar com o endereço de ip da citação.

Abreviações

Para a abreviação é usada a tag ABBR que é feita para mostrar o significado da abreviação quando o cursor estiver em cima das siglas.

Listas

Lista ordenada

Usamos a tag OL para criar uma lista ordenada e a tag LI dentro da lista para definir os itens da lista.
Também podemos definir tipos de lista com 1, A, a, I, i.

  1. Estudar HTML
  2. Estudar CSS
  3. Estudar JavaScript

Listas não ordenadas

Usamos a tag UL para criar listas não ordenas, geralmente usadas para menus e também é usada a tag LI para definir os itens.
Também podemos definir tipos de lista com disc, circle e square.

  • Camisas
  • Bermudas
  • Tênis

Listas de definição

Usamos lista de definição com a tag DL e os itens dentro dessa lista é usada a tag DT para listar o item e a tag DD para definir a explicação do item como se fosse um dicionário.
